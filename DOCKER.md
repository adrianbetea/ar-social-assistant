# Docker: LibreTranslate

## Create + start (first time)
This creates and starts the container.

```bash
docker compose up -d
```

## Start (when it already exists)
```bash
docker compose start
```

## Stop
```bash
docker compose stop
```

## Remove (if you want to recreate)
```bash
docker compose down
```

## Recreate (delete + create)
```bash
docker compose down
docker compose up -d
```
